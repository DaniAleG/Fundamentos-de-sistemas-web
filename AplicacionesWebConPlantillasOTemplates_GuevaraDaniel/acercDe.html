Acerca de la página Ejercicios

En esta página encontrarás una serie de ejercicios prácticos que te ayudarán a comprender y aplicar el uso de ciclos repetitivos (for, while y do while) en la resolución de problemas con arreglos y lógica algorítmica.

Ejercicio 1

Se requiere un algoritmo para obtener un vector (C) de N elementos que contenga la suma de los elementos correspondientes de otros dos vectores (A y B).

  • Utiliza ciclos repetitivos (for, while, do while).
  • Realiza la suma elemento a elemento.
Ejercicio 2

Se tienen los nombres de los N alumnos de una escuela, además de su promedio general. Realiza un algoritmo para capturar esta información, la cual se debe almacenar en arreglos, un vector para el nombre y otro para el promedio.

  • Captura nombres y promedios.
  • Ordena los promedios de mayor a menor.
  • Los nombres deben corresponder con los promedios.
Ejercicio 3

Cierta empresa requiere controlar la existencia de diez productos, los cuales se almacenan en un vector A, mientras que los pedidos de los clientes de estos productos se almacenan en un vector B.

  • Genera un tercer vector C que representa lo que se requiere comprar para mantener el stock de inventario.
  • Reglas:
    • Si los valores de A y B son iguales, se almacena este mismo valor.
    • Si B es mayor que A, se almacena el doble de la diferencia entre B y A.
    • Si A es mayor que B, se almacena B.
